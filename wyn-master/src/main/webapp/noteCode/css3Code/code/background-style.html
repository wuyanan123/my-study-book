背景相关样式

指定背景显示范围

使用说明：-webkit-background-clip: border(默认) / padding(padding-box);

属性值说明:

  1. border：背景范围包含边框区域
  2. padding：背景范围“不”包含边框区域
    3. 例如：background-clip: border;

指定背景显示范围练习:

默认
padding-box
border-box

指定背景图像的绘制起点

使用说明：-webkit-background-origin: padding(默认)/border(border-box)/content(content-box);

属性值说明:

  1. border：从边框的左上角开始绘制
  2. padding：从内部补白(padding)左上角开始绘制
  3. content：从内容左上角开始绘制
    4. 例如：-webkit-background-origin: border-box;

指定背景图像的绘制起点练习:

默认
border-box
padding-box
content-box

指定背景图像的尺寸

使用说明：-webkit-background-size: width height / width auto / auto height / contain / cover;

属性值说明:

  1. width height：设置的图片的宽和高
  2. contain：把原始图片在维持纵横比的前提下自动放大或缩小， 以使原始图像的宽度或者高度完全等于元素的宽度或者高度，
    3. 例如：当放大宽度先等于元素宽度时，高度会平铺图像填满元素高度
  3. cover：使原始图像在维持纵横比的前提下将背景图像自动缩放到填满元素内部， 如果元素与图像长宽比例不一致，则将多余部分裁剪掉。
    4. 例如：-webkit-background-size: 20px 40px;

指定背景图像的尺寸练习:

默认
50px 130px
contain
cover

新增用于平铺背景图像的选项

使用说明：-webkit-background-repeat: no-repeat/repeat(默认)/repeat-x/repeat-y/ space/round;

属性值说明:

  1. space：在水平、垂直平铺时，不剪掉超出的部分，也不会调整图像的尺寸， 而是自动调整图像与图像之间的间距
  2. round：自动调整背景图形的尺寸
    3. 例如：background-repeat:space;

新增用于平铺背景图像的选项练习:

默认
no-repeat
space
round
