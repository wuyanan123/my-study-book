变形处理-->3D变形

3D相关重要属性

属性：perspective,perspective-origin,transform-style:preserve-3d,backface-visibility

属性值说明:

perspective: n px; -->透视，视角
  1. 说明：perspective属性的存在与否决定了你所看到的是2次元的还是3次元的， 也就是是2D transform还是3D transform.
  2. 使用方法：两种书写形式，一种用在舞台元素上（动画元素们的共同父辈元素）； 第二种就是用在当前动画元素上，与transform的其他属性写在一起。
  3. 例子：舞台元素：.stage{perspective:300px}
    当前动画元素：.stage .box{transform:perspective(300px) rotateY(45deg)};
  4. perspective:n px;设置大小位置 相当于人的眼睛位置 设置translateZ(m px)； 当m 越小于 n 时 则子元素越小(因为离眼睛越远)；当 m 越接近 n 时 子元素 则会撑满整个屏幕； 当 m 大于 n 时 子元素会看不见，因为当眼睛后面了

属性值说明:

perspective-origin: x y ; --> 设置都是原点(也就是眼睛看是什么位置)
  1. x,y：与background-origin 值设置一样

transform-style:preserve-3d \ flat(默认值)
  1. preserve-3d: 表示3D透视,子元素将保留其 3D 位置。
  2. 默认值flat: 表示平面的,子元素将不保留其 3D 位置。

backface-visibility:
  1. 在CSS3的3D世界中，默认情况下，我们是可以看到背后的元素
  2. backface-visibility:hidden; 使后面元素不可见

3D移动

使用说明：transform:translateX(x px) translateY(y px) translateZ(z px);

属性值说明:

3D移动：
  1. translateX(x px)--->沿X轴移动
  2. translateY(y px)--->沿Y轴移动
  3. translateZ(z px)--->沿Z轴移动
  4. 例：transform:translateX(20px) translateY(100px) translateZ(40px);

3D缩放

使用说明：transform:scaleX(sX) scaleY(sY) scaleZ(sZ);

属性值说明:

3D缩放：
  1. 值说明：缩放基数为1，如果其值大于1元素就放大， 反之其值小于1，元素缩小。
  2. transform:scaleX(sx)-->scaleX(sx)表示在X轴上的缩放大小
  3. transform:scaleY(sy)-->scaleY(sy)
  4. transform:scaleZ(sz)-->表示在Z轴上缩放
    5. 例：transform:scaleZ(1.5)

3D倾斜

使用说明：transform:skew(angle,angle)

属性值说明:

3D倾斜：
  1. 使用说明：倾斜是二维变形，不能再三维空间倾斜，所以没有skewZ用法
  2. transform:skewX(angle)-->在X轴上顺时针方向倾斜
  3. transform:skewY(angle)-->在Y轴上顺时针倾斜
    4. 例：transform:skewX(30deg);transform:skewY(20deg)
