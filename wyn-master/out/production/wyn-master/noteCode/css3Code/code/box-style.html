盒的基本类型

使用说明：display:inline-block;

盒基本类型:

  1. block：块类型(例如div,p等元素)
  2. inline：行类型(例如a,span等元素)
  3. inline-block：属于block类型盒的一种，但是在显示时具有inline类型的特点。 也就是说只要将元素(无论该元素是block还是inline类型)的display：设置为 inline-block类型，该元素同时具备inline类型与block类型的特征。
  4. inline-table
  5. list-item：将多个元素作为列表来显示
  6. run-in
  7. compact

  3. auto：内容超出时，出现固定的水平 和 垂直方向的滚动条

对盒使用阴影

使用说明：box-shadow:length1 length2 length3 color;

属性值说明:

  1. length1：x轴方向偏移距离(负值向左偏移)
  2. length2：y轴方向偏移距离(负值向上偏移)
  3. length3：阴影模糊半径
  4. color：阴影颜色
    5. 例如：box-shadow:10px 10px 5px orange;
  5. inset关键字：创建盒内阴影
    例如：box-shadow:inset 0px 0px 30px orange;

指定针对元素的宽度和高度计算方法

使用说明：box-sizing:border-box / content-box(默认);

属性值说明:

  1. border-box：表示元素的宽度和高度包括补白区域(padding)与边框的宽和高
  2. content-box：表示元素的宽度和高度"不"包括补白区域(padding)与边框的宽和高
    3. 例如：box-sizing: border-box;
