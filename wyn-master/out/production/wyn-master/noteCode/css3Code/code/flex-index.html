属性说明

  1. 盒布局基本用法
    • display:-webkit-box

  2. 弹性盒布局
    1. 基本用法：display:-webkit-flex
    2. 使用弹性盒布局时相关的属性使用方法
      1. order:改变各元素的显示顺序,从小到大显示(没有设置该属性的元素优先排列在前)
      2. flex-direction:改变元素的排列方向
        1. row:横向排列(默认值)
        2. row-reverse:横向反向排列
        3. column:纵向排列
        4. column:纵向反向排列
      3. flex:单个元素使用flex属性时，元素的宽度和高度会自动扩大
      4. 多个元素使用flex属性，且值不为1， 例如设置的是三个flex:2,flex:1,flex:1;
        原理：
        纵向排列： 将空白部分按flex值总和(这里是2+1+1)分成对应份数 这里是4份，然后根据flex值分配，例如flex:2，则分配两份空白 部分的高度，分配的空白高度+元素本身的高度=元素的布局后的高度
         横向排列： 分配的空白宽度+元素本身的宽度=元素的布局后的宽度
      5. flex-grow:指定元素的宽度或高度(与flex属性的计算方法不同) 例如：容器总宽度 = 600 ，3个子元素每个元素宽度 = 150， 设置flex-grow:1,flex-grow:3,flex-grow:1;
        计算方法:
        1) 600 - 150*3(三个字段的总宽度) = 150(即空白部分总宽度)。
        2) 150 / 5(flex-grow属性值总和) = 30；
        3) 元素宽度(width样式属性值) + 30*flex-grow属性值 = 每个元素的最后宽度值
      6. flex-shrink：也是指定元素的宽度或高度
        与flex-grow的不同:
        横向排列时：如果子元素的width样式属性值的总和 小于 容器的宽度值时， 必须通过flex-grow样式属性来调整子元素宽度
        如果子元素的width样式属性值的总和大于容器的宽度值时， 必须通过flex-shrink样式属性来调整子元素宽度;
        纵向排列时：height 与 容器height 高度来 判断 ，同上

& calc方法

通过该方法来自动计算元素的宽度、高度等数值类型的样式属性值

例如:
父元素width:500px, 子元素设置为父元素的width的50%-100px;
即500px * 50% - 100px = 150px;
使用方法:
父div{
    width:500px;
}
子div{
    width:calc(50% - 100px);
}

& 使用弹性盒布局时相关的属性使用方法
  1. flex-wrap:nowrap | wrap | wrap-reverse 控制换行方式
    • nowrap:不换行
    • wrap:换行
    • wrap-reverse:虽然换行，但是换行方向与使用wrap样式属性值时相反
  2. 将flex-direction与flex-wrap合并书写在 flex-flow 样式属性值中
    例如：flex-direction:row; flex-wrap:wrap;
    flex-flow:row wrap;
  3. 指定水平方向与垂直方向的对齐方式
    1. 弹性盒布局中的一些专用术语
      • main axis:进行布局时作为布局基准的轴，横向为水平轴，纵向时为垂直轴
      • main-start / main-end:布局起点与布局终点，横为容器左右端，纵向为顶和底端
      • cross axis:与main axis垂直相交的轴，横向为垂直轴，纵向为水平轴
      • cross-start / cross-end:cross axis 轴的起点与终点
    2. justify-content 属性:用于指定如何布局容器中除了子元素之外的main axis轴上的剩余空白部分
      • flex-start: 从 main-start 开始布局所有子元素(默认值)
      • flex-end: 从 main-end 开始布局所有子元素
      • center: 居中布局所有子元素
      • space-between: 将第一个子元素布局在main-start处，最后一个布局在main-end处， 将空白部分平均分配在所有子元素与子元素之间
      • space-around: 将空白部分平均分配在一下几处，如 main-start与第一个子元素之间， 各子元素与子元素之间，最后一个子元素与main-end之间。
    3. align-items 属性与 align-self 属性
      • align-items: 用于指定子元素的对齐方式
      • flex-start: 从cross-start开始布局所有子元素
      • flex-end: 从cross-end看似是布局所有子元素
      • center: 居中布局
      • baseline: 如果子元素的布局方向与容器的布局方向不一致，该值左右等效于flex-start 保持一致时，所有子元素中的内容沿基线对齐
      • stretch: 同一行中的所有子元素高度被调整到最大。
    4. align-items 属性与 align-self 属性 的不同
      • align-items: 用于指定所有子元素的对齐方式
      • align-self: 被用于单独指定某些子元素的对齐方式
      • 例如： 将容器元素的align-items 属性值指定为center(居中对齐)后， 可以使用align-self指定第一个子元素或者其他子元素的对齐方式。
    5. align-content 属性: 进行多行布局时，使用其指定各行对齐方式。
      • flex-start: 从cross-start开始布局所有行(默认值)
      • flex-end: 从cross-end看似是布局所有行
      • center: 居中布局所有行
      • space-between: 将第一行布局在cross-start处，将最后一行布局在cross-end处， 将空白部分平均分配在各行之间。
      • space-around: 将空白部分平均分配在以下几处，如cross-start与第一行之间、 各行与各行之间、最后一行与cross-end之间。
