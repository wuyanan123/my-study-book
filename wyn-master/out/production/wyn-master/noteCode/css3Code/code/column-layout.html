多栏布局

属性列表：column-count:数字；column-gap:每个栏之间的间距； column-rule:增加间隔线；column-width:栏宽度

属性说明:

  1. column-count：属性值为一个数字表示列数，不带单位，含义是将一个元素中的内容分为多栏进行显示。
  2. column-gap：设定多栏之间的间隔距离
  3. column-rule：在栏与栏之间增加间隔线；用法与border一致；
    例：column:2px solid green;
  4. column-width：设置每一栏的宽度。
    说明：
    在设定column-width的同时必须设置盒子的width，否则盒子宽度默认为100%， 每栏宽度按照栏数平均分。盒子每栏宽度必须大于等于column-width设定的值， 否则就会减少栏数来增加每栏宽度，例如盒子宽度400px，一共2栏，那么每栏宽度就是200px， 如果设置column-width: 210px的话盒子就会变成1栏以保证每栏宽度大于等于column-width:210px;， 但是如果每栏宽度大于column-width的值时每栏宽度也不会强制等于column-width， 这么看column-width的个性有点像min-width。
